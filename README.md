## Protocolo DEVS

Es un simple protocolo para encriptar y desencriptar mensajes, utiliza una combinacion de cifrado cesar y base64.

### Como se utiliza

<h3>Paso 1: Ejecutar el archivo index, lo siguiente es la pagina principal</h3>

![Paso 1: ](/DEVS%20Protocol/imgs/Step1.png)

<br><h3>Paso 2: Escribir un mensaje</h3>

![Paso 2: ](/DEVS%20Protocol/imgs/Step2.png)

<br><h3>Paso 3: Dar clic en el boton "Codifica el mensaje", acto seguido se descargara un archivo .txt</h3>

![Paso 3: ](/DEVS%20Protocol/imgs/Step3.png)

<br><h3>Vista del archivo decodificado </h3>

![Vista ](/DEVS%20Protocol/imgs/vistaprevia.png)

<br><h3>Paso 4: Dar clic en la hipervinculo "Decodficalooo", desplegara la siguiente pagina</h3>

![Paso4: ](/DEVS%20Protocol/imgs/Step4.png)

<br><h3>Paso 5: Dar clic en el boton de "Choose File", te desplegara un cuadro de texto donde seleccionaremos el archivo previamente descargado y presionaremos el boton de "Abrir" u "Open"</h3>

![Paso5: ](/DEVS%20Protocol/imgs/Step5.png)

<br><h3>Paso 6: Una vez seleccionado el archivo, daremos clic en el boton de "Decodificar" y mostrara el archivo con el mensaje original</h3>

![Paso6: ](/DEVS%20Protocol/imgs/Step6.png)
